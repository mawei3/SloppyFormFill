Outline for NewEmployee Paste script

Import Needed modules:
    OS - Only needed if opening/creating files
    pyperclip - copy and pasting to clipboard
    email - May not be needed if parsing
    keyboard - Register hotkey, Simulate key presses - Paste > Tab > Paste > Tab etc
    Tkinter - GUI

GUI:
    Drag file to open GUI and populate forms
        Or copy to clipboard from reply
    Copy to clipboard button
    Non-GUI Paste from clipboard using hotkey

Parse EML file:
    Search EML file for each Dictionary Key and Value
    Dictionary emailDict Keys order in Ivanti (* = required) - Remove spaces at beginning and end
        FirstName * - str alpha
        MiddleInitial - str alpha
        LastName * - str alpha
        MUID * - str all digits
        DateOfBirth * (MM/DD/YYYY format) - str alpha[2] / alpha[2] / alpha[4]
        PreferredDisplayName - str alpha
        NewEmpPersonalEmail - str alpha @ alpha . alpha[3]
        DepartmentName * - str alphaNum
        Campus* - str alpha
            #Atlanta
            #Columbus
            #Douglas
            #Eastman
            #Henry
            #Law School
            #Macon
            #Newnan
            #Savannah
        Building * - str alphaNum
        Room * - str Num pref or alphaNum
        EmployeeType * - str alpha
            #Adjunct
            #Faculty
            #Other
            #Staff
        JobTitle * - str alpha
        Supervisor * - str alpha
        StartDate * (MM/DD/YYYY format) - str alpha[2] / alpha[2] / alpha[4]
        PhoneReq * (Yes/No) - str alpha Yes or No
            PhoneAccount * - str Num might have dash or periods
            UseExistingCheckBox (On by default)
                ExistingExt(On) - str Num only grab last 4
                SelectPhone(Off)
                    #Cisco 7911
                    #Cisco 7942
                    #Cisco 7945
                    #Cisco 7962
            LongDistCheckBox (Off by default)
                InternationalCheckBox (Off by default)
            VoiceMailCheckBox (Off by default)
            CallForwardCheckBox (Off by default)
        CanvasCheckBox (On by default)
        MyMercerCheckBox (Off by default)
        CampusNexusCheckBox (Off by default)
            CopyNexusFrom * - str alpha[space]alpha
        CampusVueFPCheckBox (Off by default)
        NetworkDriveCheckBox (Off by default)
            NetworkDrivePath * - str prefered \\<servername>\department
        VPNCheckBox (Off by default - No Form to submit/Add to Notes if possible if VPN is requested)
            VPNFacStaff *
                #Faculty
                #Staff
        Live25Checkbox (Off by Default)
        ComputerRequirements *
            New computer has been/will be ordered
            No computer required
            Thin client has been/will be ordered
            Use existing computer
                ComputerRTNumber * - str alphaNum
            Using existing thin client
                ExistingThinClientName * - str alphaNum
        AdditionalNotes - str block text

        # Character (#) Denotes variable options Do not count for dictionary index

Paste from dictionary to Website
    Select first field
    For loop emailDict[0] to emailDict[43]
        Grab Value of Key -> Paste into field > Tab to Next field

REGEX

\bGUI[\r\n]+([^\r\n]+)
\bFirstName(.+)[\n\r] Entire String on line
(.+)[\n\r] Grabs next line

Create installer

pyinstaller.exe --onefile --windowed app.py
